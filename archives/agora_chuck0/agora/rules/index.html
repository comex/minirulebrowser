A Note on Rule Updates and Locking

When I am in the process of updating the Rules, the Rules on the web are "locked" and you will not be able to access them. This is for two reasons.

  1. I cannot guarantee that the programs used to give access to the Rules over the web will work properly when I am in the middle of updating the Rules

  2. Even if the programs did work properly, I do not want to mislead people by making available a Ruleset which has incorporated some, but not all, of a set of Rule changes which took place at the same time.

When there is a set of changes to be made to the Rules, I try to do them in a single sitting, in which case they should not take more than an hour, or maybe two. However, if there are an exceptionally large number of Rule changes to record, it may take a few different sessions over the course of a day or so in order to update the Rules, during which time the Rules will remain locked. I apologize for the inconvenience, but I feel strongly about the reasons I listed above, particularly the second.
